# Desafio DataX — Análise Exploratória de Clientes

## Visão geral

Projeto de análise exploratória de dados aplicado a um cenário de telecomunicações, com foco em compreender padrões de comportamento de clientes e gerar recomendações de negócio orientadas por dados.

## Objetivo

- Explorar a base de clientes.
- Investigar relações entre variáveis contratuais, financeiras e de serviços.
- Produzir visualizações e insights interpretáveis.
- Apoiar recomendações para melhoria de retenção e relacionamento com clientes.

## Tecnologias utilizadas

- Python
- pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook

## Arquivo principal

O notebook `Desafio Data_X.ipynb` contém a análise completa, incluindo preparação dos dados, gráficos e recomendações.

## Principais frentes da análise

- Leitura e preparação da base de dados.
- Comparação entre perfis de clientes.
- Visualização de variáveis relevantes.
- Interpretação de padrões por contrato, tempo de relacionamento, tipo de serviço e forma de pagamento.
- Consolidação de recomendações de negócio.

## Como executar

### Google Colab

1. Faça upload do arquivo de dados do projeto.
2. Abra o notebook `Desafio Data_X.ipynb`.
3. Execute as células em sequência.

### Ambiente local

```bash
git clone https://github.com/Willserra78/Desafio_DataX.git
cd Desafio_DataX
pip install pandas numpy matplotlib seaborn
jupyter notebook
```
